# Cookie Sales Analysis Dashboard

## Overview
This project is a Power BI dashboard designed to analyze cookie sales performance using interactive data visualization and calculated measures.

The dashboard provides insights into revenue, cost, profit, and product performance to help understand sales trends and profitability.

## Dashboard Features
The report includes several key metrics and visualizations:

- Total Revenue
- Total Cost
- Total Profit
- Profit Margin
- Number of Customers
- Orders exceeding specific sales thresholds

## Data Analysis
The dashboard uses calculated measures and functions in Power BI to derive business insights, including:

- Revenue calculation
- Cost aggregation
- Profit calculation
- Profit margin percentage
- Product-level performance analysis
- Sales trends by weekday

## Key Insights
The dashboard allows users to:

- Identify the most profitable cookie products
- Compare revenue and cost across products
- Analyze sales patterns across different days of the week
- Evaluate profit margins

## Tools
- Power BI
- Data modeling
- DAX measures
- Data visualization
